## 💻 Projects

- **[Libranova](https://github.com/SalihMezraoui/Libranova)** · [🔗 Live Demo](https://libranova-lac.vercel.app) — Full-stack library management system (Bachelor's thesis). **20 REST endpoints** for book search, borrowing, reviews, wishlist & admin management. **Auth0** (OAuth2/JWT), role-based access with Spring Security, **Stripe** payments, i18n (DE/EN), **65 automated tests** (JUnit 5 / Mockito). CI/CD via GitHub Actions → Vercel (frontend) + Railway (backend + MySQL), Dockerized.
  <br/>*Spring Boot · React · MySQL · Docker · Auth0 · Stripe · GitHub Actions · Vercel · Railway*

- **[ShopNook](https://github.com/SalihMezraoui/ShopNook)** — Full-stack e-commerce platform with Angular & Spring Boot. Custom + Spring Data REST endpoints, **Okta** (OAuth2/JWT), Stripe Payment Intent API, bidirectional pagination, product search across 11 categories.
  <br/>*Spring Boot · Angular · MySQL · Stripe · Okta · JWT*

- **[name-pad](https://github.com/SalihMezraoui/name-pad)** — Name management web app: search, filtering (gender, prefix, suffix, syllable count) and pagination. Regex-based matching and CSV import into MongoDB.
  <br/>*Node.js · Express · MongoDB*

- **[study-path](https://github.com/SalihMezraoui/study-path)** — Student web platform with planner, forum and learning resources.
  <br/>*PHP · HTML/CSS/JS*
